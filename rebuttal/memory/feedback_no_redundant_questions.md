---
name: no_redundant_questions
description: Don't ask clarifying questions when the answer is already obvious or data already exists — just do it
type: feedback
---

When the user asks for something and the data is already available, just show it. Don't ask redundant clarifying questions.

**Why:** User got frustrated when I asked "do you want to see X?" when X was clearly what they asked for and was already computed.

**How to apply:** If the request is clear and the data exists, execute immediately. Only ask when genuinely ambiguous.
